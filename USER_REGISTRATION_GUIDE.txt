================================================================================
USER MANAGEMENT FEATURES
================================================================================

ADDING USERS:
- No limit on number of accounts
- Each user gets unique email login
- Temporary password set by admin
- User changes password on first login

EDITING USERS:
- Change role (Navigator to Coordinator)
- Update assigned locations
- Enable/disable 2FA
- Reset passwords
- Activate/deactivate accounts

DELETING USERS:
- When staff leaves, admin can:
  * Soft delete (deactivate account, keeps history)
  * Hard delete (removes completely)
- All activity history preserved for audit

BULK OPERATIONS:
- Export user list to Excel
- Import users from CSV (future feature)
- Bulk deactivate at end of contract

================================================================================
ACCESS CONTROL BY ROLE
================================================================================

WHAT EACH ROLE SEES:

NAVIGATOR (Basic Staff):
- Client Management
- Health Navigation
- Local Demographics
- GP Engagement
- Program Resources
- NO reporting
- NO user management

COORDINATOR (Team Lead):
- Everything Navigator has PLUS:
- Dashboard
- Program Reporting
- Workforce Tracking
- AI Insights
- NO unified reporting
- NO user management

ADMINISTRATOR (IT/System):
- Everything PLUS:
- Unified Reporting
- User Management
- Activity Logs
- System Settings

================================================================================
SECURITY FEATURES
================================================================================

1. PASSWORD REQUIREMENTS:
- Minimum 8 characters
- Must change on first login
- Password reset via email

2. TWO-FACTOR AUTHENTICATION:
- Optional per user
- SMS or email code
- Admin can enforce for all users

3. ACTIVITY TRACKING:
- Every login tracked
- All actions logged
- Export logs for audit
- Track by user and location

4. SESSION MANAGEMENT:
- Auto logout after inactivity
- Single session per user
- Force logout option
